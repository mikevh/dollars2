# Project Overview

Dollars2 is a zero-based budgeting web app modeled after EveryDollar. Every dollar of income is assigned to a budget category until remaining equals $0. Personal use initially but designed for multi-user.

**Tech stack:** React (Tailwind CSS), .NET backend, MSSQL database, self-hosted

**Motivation:** EveryDollar functionality with self-hosted control and dual bank sync providers (Plaid free tier + SimpleFIN at $1.50/mo).

All design decisions reference EveryDollar as the UX baseline. Keep the app simple — no analytics, no debt tracking, no auto-categorization.
